# InternReady AI

InternReady AI is a Streamlit-based career roadmap tool for students preparing for internships, projects, and skill development.

## Features

- Personalized roadmap generation
- Domain-specific skills and project ideas
- Learning resources
- Internship readiness score
- Personalized advice
- Downloadable roadmap
- Progress checklist
- Resume helper
- LinkedIn caption helper
- GitHub presentation tips

## Tech Stack

- Python
- Streamlit
- Rule-based recommendation logic
